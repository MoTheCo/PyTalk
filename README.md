# PyTalk
a simple command line based tts 

How to use it ?


Linux/Mac 

1. Open a Terminal 
2. cd /to/the/path/of/the/script
3. pip3 install -r ./Requirements.txt
4. Python3 ./tts.py

Windows

1. Open a command prompt 
2. cd \to\the\path\of\the\script
3. pip3 install -r ./Requirements.txt
4. Python3 ./tts.py
